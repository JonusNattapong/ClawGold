# Prompt Templates

This directory contains prompt templates for AI research tools.

## Available Templates

### market_outlook.txt
General market outlook and sentiment analysis.

### technical_analysis.txt
Technical analysis with indicators and chart patterns.

### news_sentiment.txt
News sentiment analysis and impact assessment.

### trade_setup.txt
Concrete entry, stop-loss, and take-profit trade construction.

### risk_management.txt
Position sizing, drawdown limits, and risk controls review.

### macro_event_impact.txt
Impact analysis for macroeconomic events and volatility windows.

### scenario_analysis.txt
Base/bull/bear scenario planning with probabilities and triggers.

## Usage

Templates use placeholder variables:
- `{symbol}` - Trading symbol (e.g., XAUUSD)
- `{query}` - User's research query

## Adding New Templates

1. Create a new `.txt` file
2. Use `{symbol}` and `{query}` placeholders
3. Add description to this README
